# Azure Week 4①：Azure BastionとVPN（セキュアな城への道）

前章（Week 3）で、NSGによって「外からは一切アクセスできない鉄壁のVNet（城の中）」を作りました。
しかし、それでは「管理者である自分自身（あなた）」もサーバーの中にログイン（RDPやSSH）してメンテナンス作業をすることができません。
自分だけが安全に中に入るための「秘密の抜け道」の設計が、Week 4の主題です。

## 1. 危険なアンチパターン：パブリックIPの付与
「私がログインするためだけに、VMに一瞬だけパブリックIPアドレス（グローバルIP）を付けて、ネット越しにポート3389(RDP)を開けちゃおう！」
➡️ **数十分以内に、海外のハッカーのボットから「総当たり（ブルートフォース）攻撃」を受け、パスワードを突破されて仮想通貨のマイニングウイルスを仕込まれ終了します。**
（試験でも実務でも、本番環境のVMにパブリックIPを持たせることは極刑に値します）。

## 2. 現代の正解「Azure Bastion (要塞・踏み台)」

パブリックIPを持たない「完全に内側に引きこもったVM」に対して、安全にログインするためのPaaS機能が **Azure Bastion** です。

*   **仕組み:** あなたがブラウザでAzureポータルにアクセスし、「Bastion経由で接続」ボタンを押すと、Azureが裏側に用意した踏み台サーバー（Bastionホスト）を経由し、**ブラウザのHTML5画面上でVMのデスクトップ画面を操作**できます。
*   **利点:**
    1.  VM側にパブリックIPが一切不要（完全に隠蔽されたまま）。
    2.  あなたのPCに、面倒なVPNソフトを入れる必要が一切ない。
    3.  NSG等でSSH用ポート(22)やRDP用ポート(3389)をインターネットに向けて開ける必要がない（Bastionの中継ポートだけ開ければOK）。

> [!CAUTION]
> **🚨 Exam Trap (Bastion用サブネットの制約)**
> バスティオンを作成するには、VNetの中に彼ら専用の「控室（サブネット）」を用意してあげる必要があります。
> 試験で問われるのは、そのサブネットの名前です。
> 
> **正解:** **大文字・小文字・スペルが一言一句間違っていない `AzureBastionSubnet` という名前のサブネットを事前に作っておかなければ、Bastionは絶対に作成できません。**
> （※「BastionSubnet」などの適当な名前はエラーで弾かれます）。

## 3. 本格的な回線（VPN Gateway vs ExpressRoute）

Bastionは個人のPCから入るためのものでした。
では、「東京本社の自社ビル（オンプレミス）のネットワークと、AzureのVNetを丸ごと結合して、太いパイプでつなぎたい」場合はどうするでしょうか？
AZ-104では、企業用の2大回線に関する特性の違いが必ず問われます。

### ① VPN Gateway (サイト間VPN / S2S VPN)
*   **経由:** 一般の（皆がYouTubeを見ている）**パブリック・インターネット回線**の上を通ります。
*   **セキュリティ:** インターネットを通るため、中身を暗号化（IPsecのトンネルによるカプセル化）して泥棒の目から隠します。
*   **メリット:** 安い。すぐ数十分で構築できる。
*   **デメリット:** 回線の状態によって遅延（PingPing）がブレる。「最大1.25 Gbps」程度の速度が限界。

### ② ExpressRoute (エクスプレス・ルート)
*   **経由:** NTTやKDDIなどのプロバイダが用意した「あなた専用の物理的な光ファイバーケーブル」を利用します（**インターネットを一切通りません**）。
*   **セキュリティ:** 完全に専用のトンネルを通るため、**デフォルトでは暗号化されていません（暗号化する必要すらないから）**。
*   **メリット:** 回線速度が保証され、遅延が極小かつ最強に安定。（最大100 Gbpsの超弩級の太さ）。
*   **デメリット:** アホみたいに高い。工事が必要なので、開通までに数ヶ月かかることもある。

> [!IMPORTANT]
> **🚀 Exam Focus (アーキテクトの決断)**
> 試験で「要件として、『絶対に確実で一貫性のあるレイテンシ（低遅延）』と『最高速度』が求められています。また『インターネット回線を一切関与させないこと』が条件です。どちらを選ぶべきですか？」と問われたら、答えは **ExpressRoute** の一択です。
