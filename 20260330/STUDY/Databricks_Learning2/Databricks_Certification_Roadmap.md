# Databricks Certified Data Engineer Associate 完全攻略ロードマップ

本番環境（実務）でのアーキテクチャ設計能力は「Databricks_Learning (Part 1)」ですでに身につけています。
この `Databricks_Learning2` は、**「試験特有の嫌らしいひっかけ問題」や「厳密なSQL/Python構文の暗記」** に特化し、確実に70%以上のスコアをもぎ取るためのハック・プログラムです。

## 🎯 試験の概要と配点
認定試験は「制限時間 90分」「多肢選択式（4択や複数選択）」「全45問（※採点対象外の問題を含む場合あり）」で行われます。合格ラインは **70%** です。

以下の比率で出題されるため、この割合に合わせて4週間のプログラムを組み込んでいます。

1. **Databricks Workflows (タスクオーケストレーション)** - 10%
2. **Databricks ワークスペースアーキテクチャ / セキュリティ** - 10% -> *(Week 1で攻略)*
3. **データマネジメント (Delta Lakeの仕様・最適化)** - 20% -> *(Week 2で攻略)*
4. **Spark SQL・PySpark を用いた ELT (データ変換)** - 30% -> *(Week 3で攻略)*
5. **Delta Live Tables と Structured Streaming** - 15% -> *(Week 4で攻略)*
6. **データ・ガバナンス (Unity Catalog)** - 15% -> *(Week 4で攻略)*

## 🚦 1ヶ月プログラムの進め方
この試験は「**理解しているか？**」ではなく「**正しい引数や設定画面の動きを丸暗記しているか？**」を問う問題が多発します。
例えば、「プロダクション環境でバッチを動かすべきクラスターは？」という実務の常識から、「空のデータを削除するDataFrame APIの正しいメソッドは？(`.dropna()` なのか `.drop_nulls()`なのか)」といった構文問題まで出題されます。

各WeekのMarkdownファイルには、
**「試験ではどう問われるか（Exam Focus）」**
**「絶対にはまってはいけない罠（Exam Trap）」**
という専用のアラートを多数仕掛けています。

各ドキュメントを読みながら「なるほど、問題文でこのキーワードが出たら、答えはこれだな」という**フラグ回収のゲーム**感覚で読み進めてください。それでは、Week 1の「コンピュートとWorkflows」から攻略を開始しましょう！
