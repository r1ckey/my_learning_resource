# AZ-104 Azure Administrator 攻略ロードマップ (前半戦)

データエンジニアとして「データ基盤（Databricks等）」を作るにあたり、**「その基盤を載せるクラウドのネットワークやセキュリティはどうなっているのか？」**を理解しないままでは、絶対に設計・運用に行き詰まります。

このPart 1〜Part 2では、Microsoft Azureのインフラ基盤全てを司る中核管理者資格 **【AZ-104】** の内容をマスターし、セキュアなインフラ構築能力を身に付けます。

## 🎯 AZ-104: 試験の概要と配点
AZ-104は「暗記」にとどまらず、**「要件を満たしつつ、一番安くて安全な構成を選ぶ能力（アーキテクチャ設計）」**が問われる実務直結の超重要資格です。合格ラインは700点（約70%）です。

1. **AzureのIDとガバナンス (15-20%)** -> *(Week 1で攻略)*
   * Entra ID（旧Azure AD）、RBAC、Azure Policy、リソースロック
2. **ストレージの実装と管理 (15-20%)** -> *(Part 3のDP-203へ統合)*
3. **Azureリソース（コンピューティング） (20-25%)** -> *(Week 2で攻略)*
   * 仮想マシン(VM)、VM Scale Sets、App Service、Functions
4. **仮想ネットワーキングの実装と管理 (20-25%)** -> *(Week 3で攻略)*
   * 第1関門。VNet、サブネット、ピアリング、NSG、VPN、踏み台
5. **Azureリソースの監視と保守 (10-15%)** -> *(Week 4で攻略)*
   * Azure Monitor、Log Analytics、Backup、Site Recovery

## 🚦 学習の進め方
Databricksの資格試験の時と同じように、**「🚨 Exam Trap（試験のひっかけ）」** や **「🚀 アーキテクトの判断基準」** を豊富に交えてドキュメント化しています。

特にAZ-104は「2つの似たような機能（オンプレADとEntra ID、RBACとEntraロールなど）の、絶対に機能が重ならない違い」を突いてきます。
「このキーワードが出たらこっちが正解！」という感覚を掴みながら、Azureインフラの頂点を目指しましょう！
